The GridEditorSite class-interface for access the grid cell internals. It may be necessary if you are coding a custom cell editor. For example, it's possible to get the dimensions of the cell, the index model - QModelIndex, show a tooltip near the cell and etc. More...

Detailed Description

The GridEditorSite class-interface for access the grid cell internals. It may be necessary if you are coding a custom cell editor. For example, it's possible to get the dimensions of the cell, the index model - QModelIndex, show a tooltip near the cell and etc.
