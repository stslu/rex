Detailed Description

GridDataTableColumn represents the base class of the grid column that has a link to the data via dataBinding property.

enum GridEditorType

This enumerator describes the possible types of grid cell editors for table view, banded table view, card view.

Qtitan::GridTableColumn* column = (Qtitan::GridTableColumn *)view->getColumnByModelColumnName("Photo");
column->setEditorType(Qtitan::PictureEditorType);
Qtitan::GridPictureEditorRepository* pictureEditor = (Qtitan::GridPictureEditorRepository *)column->editorRepository();
pictureEditor->setAspectRatioMode(Qt::KeepAspectRatio);
ConstantValueDescription
AutoEditorType0Automatic selection of the editor by edited value. By default.
StringEditorType1The editor of an unary line. See GridStringEditorRepository for the details.
MemoEditorType2The editor of multi-line text. See GridMemoEditorRepository for the details.
ComboBoxBlobEditorType3The editor of multi-line text in popup window. See GridBlobEditorRepository for the details.
NumericEditorType4The editor of numerical value. See GridNumericEditorRepository for the details.
CheckBoxEditorType5The editor of boolean value. See GridCheckBoxEditorRepository for the details.
ComboBoxEditorType6The editor allows to show combo boxes in the grid cell. See GridComboBoxEditorRepository for the details.
TableEditorType7The editor allows to specify the other datagrid as details for the editing value. Details grid will be shown in the popup window by clicking on the button that embeded to the grid cell. See GridTableEditorRepository for the details.
DateTimeEditorType8The editor of DateTime value. See GridDateTimeEditorRepository for the details.
TimeEditorType9The editor of Time value. See GridTimeEditorRepository for the details.
DateEditorType10The editor of Date value. See GridDateEditorRepository for the details.
PictureEditorType11The editor of picturies or images. See GridPictureEditorRepository for the details.
ComboBoxPictureEditorType12The editor of picturies and images. The pucture will be edited in the popup window like combobox. See GridPictureComboBoxEditorRepository for the details.
DelegateAdapterEditorType13The type of editor which allows you to use delegates from the regular table QTableView in the QtitanDataGrid. For more details see GridDelegateAdapterEditorRepository or follow this topic - QtitanDataGrid cell editor
ProgressEditorType14The editor for showing and editing numeric value. Custom ProgressBar widget is used for this purposes. See GridProgressEditorRepository for the details.
CustomEditType15The editor of value creating by cutomer. Please read this article for more details - QtitanDataGrid cell editor
