Detailed Description

The GridBandedTableViewOptions is a class of the settings for the banded view.

Contains methods and properties to control the banded view behavior. The class is automatically created within the view and is accessible via optons() function. Please note that some of the settings that relate to view layout can be saved and restored using the mechanism save/restore layout the to XML.
