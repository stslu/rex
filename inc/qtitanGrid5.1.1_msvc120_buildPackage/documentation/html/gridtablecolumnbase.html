enum ColumnFixedKind

Enumerator describes the side of the view for fixed columns.

ConstantValueDescription
FixedNone0Column should not be fixed (horizontally scrolled). Default value for new columns.
FixedLeft1Column should be fixed at the left side of the grid view.
FixedRight2Column should be fixed at the right side of the grid view.
